# ATS-friendly-resume-builder-Features

Live Preview — Resume updates in real-time as you type
ATS Score — Instant score out of 100 with actionable feedback
Sections — Basics, Experience, Education, Skills
PDF Download — Print-optimised layout (Ctrl+P or Download button)
No backend needed — Runs entirely in the browser

ATS Tips Built In

Match job description keywords in your Summary and Skills
Use plain bullet points (no tables, columns, or icons)
Keep font standard and content clean for parser compatibility
